## ✨ Features

- 🎨 **Premium Design** — Glassmorphism UI, smooth animations, and gradient accents
- 🌗 **Dark / Light Mode** — Theme switcher with `localStorage` persistence
- 🎨 **Color Themes** — 5 accent color options (blue, pink, orange, green, grey)
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile
- 📊 **Live GitHub Stats** — Dynamic contribution charts and language breakdowns
- 🏆 **Animated Counters** — Auto-counting stats for plugins, packages, and downloads
- 📝 **Blog System** — Markdown-powered blog with modal reader
- 🖼️ **Portfolio Lightbox** — Interactive project showcase with iframe previews
- 📧 **Contact Form** — Web3Forms integration with live validation
- ♿ **Accessible** — Skip nav, ARIA landmarks, keyboard navigation, semantic HTML
- 🔍 **SEO Optimized** — Meta tags, Open Graph, JSON-LD structured data, sitemap

## 🏃 Local Development

```bash
# Clone the repository
git clone https://github.com/Mirjan-Ali-Sha/mirjanalisha.github.io.git
cd mirjanalisha.github.io

# Serve locally (any static server works)
npx serve .

# Or use Python
python -m http.server 8000

# Or use VS Code Live Server extension
```

Visit `http://localhost:3000` (or your server's port).
